Grouping and Metadata

In this lab we will learn how we can group devices and object together and how we can tag devices and objects to make it easier to group

Create Groups

  1. Log into the NMS
  2. Go to Devices -> Grouping -> Device Groups
  3. Click Add Device Group
  • a. Name: YOURNAME
  1. Save
  2. Click Add Rule a. Device Name: YOURNAME
  3. Save
  4. Click Apply Rules

  1. Go to Devices -> Grouping -> Object Groups
  • a. Click ‘Add’ -> Class
  1. Name: YOURNAME
  2. Save
  3. Click Add -> Group
  • a. Name: YOURNAME - CPU
  1. Save
  2. Click Add Rule
  • a. Device Group: YOURNAME
  • b. Object Name: ^CPU\d+$
  • c. Plugin: SNMP -d. Object Type: CPU
  1. Sav

  1. Click Add -> Group
  • a. Name: YOURNAME – Important Disks
  1. Save
  2. Click Add Rule
  • a. Object Name: .*
  • b. Plugin: SNMP
  • c. Object Type: Disk
  1. Save

Create Metadata

  1. Log into the NMS
  2. Go to Administration -> Metadata Schema
  3. Click on Add Namespace
  • a. Name: YOURNAME
  1. Save
  2. Select your namespace and click Add Attribute
  • a. Target: object
  • b. Type: text
  • c. Name: Priority
  1. Save

  1. Go to Devices -> Object Manager
  2. Click Clear Filter
  3. Search ‘1/1/0/0’
  4. Find these two objects

  1. For each of them
  • a. Click on the metadata icon
  • b. Find your namespace and your attribute
  • c. Value: Yes
  1. Update and save

  1. Search for Fa0/2/2
  • a. Select any two interfaces that you want and change the metadata namespace value to Yes, like on the step above

  1. Go to Devices -> Grouping -> Object Groups
  2. Select your class
  3. Click Add -> Group
  • a. Name: YOURNAME – Priority Interfaces
  1. Save
  2. Click Add Rule
  • a. Plugin: Any
  • b. Object Type: All Object Types
  • c. Metadata Namespace: YOURNAME
  • d. Metadata Attribute: Priority
  • e. Metadata Value: Yes
  1. Save

  1. Click Apply Rules

Review These groups will be used for a later lab, therefore there is nothing to review on this stage.
